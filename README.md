# Shopwave
A fully responsive, single-page e-commerce storefront built with pure HTML, CSS, and JavaScript — no frameworks, no dependencies.

✨ Features

1.Product grid with 12 items across 4 categories (Clothing, Footwear, Accessories, Electronics)
2.Category filters to dynamically browse products
3.Slide-out cart drawer with quantity controls and live total calculation
4.Wishlist toggle on each product card
5.NEW / SALE badges with original vs. discounted pricing
6.Star ratings & review counts on every product
7.Toast notifications for add-to-cart and wishlist feedback
8.Sticky glassmorphism navbar with live cart count
9.Fully responsive layout for mobile and desktop
